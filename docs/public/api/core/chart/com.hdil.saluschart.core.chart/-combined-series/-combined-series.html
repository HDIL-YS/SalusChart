chart/com.hdil.saluschart.core.chart/CombinedSeries/CombinedSeries

CombinedSeries

constructor(type: CombinedSeriesType, data: List<BaseChartMark>, color: Color = Color.Unspecified, axis: CombinedAxis = CombinedAxis.LEFT, label: String = "", strokeWidth: Float = 4.0f, showPoints: Boolean = false, areaAlpha: Float = 0.25f, pointType: PointType = PointType.Circle, pointRadius: Dp = 4.dp, barWidthRatio: Float = 0.8f, barCornerRadiusFraction: Float = 0.0f)

Parameters

type

How this series is drawn (LINE / BAR / SCATTER / AREA / RANGE_BAR).

data

Index-aligned marks. Use ChartMark for LINE/BAR/SCATTER/AREA and RangeChartMark for RANGE_BAR (its RangeChartMark.minPoint/RangeChartMark.maxPoint define the bar span).

color

Series color. Color.Unspecified falls back to the chart palette at draw time.

axis

Which Y-axis this series is measured against (CombinedAxis.LEFT or CombinedAxis.RIGHT).

label

Legend / tooltip label for this series.

strokeWidth

LINE/AREA: line thickness in pixels.

showPoints

LINE/AREA: also draw a marker at each point. (SCATTER always shows markers.)

areaAlpha

AREA: opacity of the filled region below the line (0..1).

pointType

LINE/SCATTER marker shape.

pointRadius

LINE/SCATTER marker radius.

barWidthRatio

BAR/RANGE_BAR: bar width as a fraction of its slot (0..1).

barCornerRadiusFraction

BAR/RANGE_BAR: corner rounding as a fraction of the bar half-width.
